# 划词英译中 + 生词高亮

一款简洁高效的 Chrome 浏览器扩展，支持网页英文划词翻译、生词自动高亮、生词本管理，帮助你在日常阅读中自然积累词汇。

![Chrome Extension](https://img.shields.io/badge/Chrome-Extension-blue) ![Manifest V3](https://img.shields.io/badge/Manifest-V3-green) ![Version](https://img.shields.io/badge/Version-1.0-orange)

## 功能特性

### 划词翻译
- 鼠标选中英文单词或短语，自动弹出翻译窗口
- 支持单击选中翻译、双击单词翻译
- 翻译结果展示原文和中文释义
- 翻译缓存，减少重复请求

### 生词本系统
- 点击收藏图标，一键将单词加入生词本
- 再次点击取消收藏
- 生词数据本地持久化存储，无需登录

### 生词高亮
- 页面自动扫描并高亮显示生词本中的单词
- 鼠标悬停高亮单词，直接显示翻译（无需调用API，响应更快）
- 高亮颜色、下划线样式可自定义配置
- 支持深色模式

### 网站黑名单
- 在设置页面管理黑名单域名
- 黑名单内的网站自动禁用划词翻译和生词高亮

### 多翻译引擎
| 引擎 | 说明 | 鉴权方式 |
|------|------|----------|
| 百度翻译（通用） | 基于百度通用翻译API | App ID + 密钥 |
| 百度翻译（大模型） | 基于百度大模型翻译API，更精准自然 | App ID + API Key |
| LibreTranslate | 开源免费翻译服务 | 无需密钥 |

### 数据管理
- 导出生词本为 JSON 文件
- 导入生词本（支持合并去重）
- 清除翻译缓存
- 清空生词本

## 安装方法

### 开发者模式安装（推荐）

1. 下载或克隆本项目到本地
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择项目文件夹
6. 扩展安装完成，图标将显示在浏览器工具栏

## 配置翻译API

### 百度翻译（通用）

1. 访问 [百度翻译开放平台](https://fanyi-api.baidu.com) 注册账号
2. 在「开发者信息」页面获取 **App ID** 和 **密钥**
3. 开通「通用翻译API」服务
4. 在扩展设置页面填入 App ID 和密钥

### 百度翻译（大模型）

1. 访问 [百度翻译开放平台](https://fanyi-api.baidu.com) 注册账号
2. 获取 **App ID**（通用翻译和大模型翻译共用）
3. 在「API Key管理」页面创建 **API Key**
4. 开通「大模型翻译API」服务
5. 在扩展设置页面选择「百度翻译（大模型）」
6. 填入 **App ID** 和 **API Key**

### LibreTranslate

无需配置，开箱即用（公共实例可能有访问限制）。

## 使用方法

### 基本翻译

1. 在任意网页上选中英文文本
2. 等待短暂延迟后，翻译窗口自动弹出
3. 窗口显示原文和中文翻译

### 收藏单词

1. 翻译窗口右上角有收藏图标 ☆
2. 点击图标，单词加入生词本，图标变为 ★
3. 再次点击取消收藏

### 生词高亮

1. 已收藏的单词会在后续浏览中自动高亮显示
2. 鼠标悬停高亮单词，直接显示翻译小窗
3. 高亮颜色可在设置中自定义

## 设置选项

访问扩展设置页面（右键扩展图标 → 选项），可配置：

| 设置项 | 说明 |
|--------|------|
| 划词翻译 | 开启/关闭划词翻译功能 |
| 触发延迟 | 选中文本后等待的时间（0-1000ms） |
| 字体大小 | 翻译弹窗字体大小 |
| 启用高亮 | 开启/关闭生词高亮 |
| 高亮颜色 | 自定义高亮背景色 |
| 显示下划线 | 为高亮单词添加下划线 |
| 翻译API | 选择翻译引擎 |
| 黑名单管理 | 添加/移除禁用翻译的域名 |

```
│   ├── content.js         # 内容脚本：划词监听、生词高亮

├── popup/
│   ├── popup.html         # 弹出面板页面

│   └── popup.css          # 弹出面板样式
├── options/
│   ├── options.html       # 设置页面
│   ├── options.js         # 设置页面逻辑
│   └── options.css        # 设置页面样式
├── wordbook/
│   ├── wordbook.html      # 生词本页面

│   └── wordbook.css       # 生词本样式

```

## 技术栈

- **HTML + CSS + JavaScript** — 纯原生实现，无框架依赖
- **Chrome Extension Manifest V3** — 最新扩展规范
- **chrome.storage.local** — 本地持久化存储
- **Service Worker** — 处理翻译API请求，解决跨域问题

## 隐私说明

- 所有生词数据存储在浏览器本地，不会上传到任何服务器
- 翻译请求通过 Service Worker 代理发送到翻译API
- 不收集任何用户浏览数据
